# Portfolio Website

## Getting Started

To run this project locally, follow these steps:

1.  **Open in VS Code**: Open the `portfolio-website-main` folder in Visual Studio Code.
2.  **Open Terminal**: Press `Ctrl` + `~` (tilde) or go to `Terminal` > `New Terminal`.
3.  **Install Dependencies**: Run the following command if you haven't already:
    ```bash
    npm install
    ```
4.  **Start Development Server**: Run:
    ```bash
    npm run dev
    ```
5.  **View in Browser**: 
    - Hold `Ctrl` and click the link shown in the terminal (e.g., `http://localhost:8080/`).
    - Or open your browser and navigate to that address manually.

## Project Structure

- `src/components`: Contains all React components (Hero, About, Projects, etc.).
- `src/pages`: Main page layouts.
- `public/Images`: Place your images here.
